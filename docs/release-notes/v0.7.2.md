# v0.7.2

## Added

**Connection secret on `Cdn`** — the delivery endpoint published for app and
future-DNS wiring (keys: `technical_domain`, `url`; public data, no
credentials):

```yaml
spec:
  forProvider:
    origin:
      bucketRef: { name: site-assets }
      https: true
  writeConnectionSecretToRef:
    name: site-assets-cdn
```

```sh
kubectl get secret site-assets-cdn -o jsonpath='{.data.url}' | base64 -d
# → https://<id>.cdn.twcstorage.ru
```

Field reference:
[docs/cdn.md](https://github.com/lebedevdsl/crossplane-provider-timeweb/blob/v0.7.2/docs/cdn.md).
